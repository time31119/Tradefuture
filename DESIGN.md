# TradeFuture DApp - Design System

## Product Identity

TradeFuture is a decentralized BTC prediction market with an insurance pool protocol on BSC. The design language merges **crypto terminal aesthetics** with **premium gold trading** vibes.

## Design Concept: "Golden Vault"

**Core Imagery**: A high-end trading terminal viewed through tinted amber glass. Think Bloomberg Terminal meets a luxury watch boutique - precision data display wrapped in warm metallic tones against deep black backgrounds.

**Physical Anchor**: The interior of a private vault trading room - matte black surfaces, brushed gold fixtures, recessed amber lighting. Every element feels deliberate, weighted, and valuable.

## Color System

| Token | Value | Usage |
|-------|-------|-------|
| Background | `#0B0E14` | Page base - deep obsidian black |
| Surface | `#141A24` | Card/container - gunmetal with blue undertone |
| Surface Elevated | `#1C2333` | Hovered/active surfaces |
| Primary | `#F5A623` | Gold - buttons, highlights, key data |
| Primary Light | `#F7C948` | Gradient endpoint, secondary highlights |
| Success | `#00C897` | Bullish/gain/profit - emerald green |
| Danger | `#FF6B6B` | Bearish/loss - coral red |
| Text Primary | `#FFFFFF` | Headlines, primary values |
| Text Secondary | `#8896A6` | Descriptions, labels, muted data |
| Border | `#2A2F3F` | Dividers, card edges |
| Border Active | `#F5A623` | Active state borders (gold glow) |

## Typography

- **Data/Numbers**: System monospace for trading values, large and bold
- **Headlines**: Bold, tight tracking - authority and precision
- **Labels**: Small caps feel, generous letter spacing - HUD-like
- **Body**: Clean, readable - not too small for mobile

## Visual Techniques

1. **Gold Glow**: Cards and active elements use amber/gold box-shadow for warmth
2. **Gradient Accents**: `#F5A623` → `#F7C948` for primary CTAs
3. **Subtle Borders**: 1px borders with low-opacity gold on active elements
4. **Depth via Shadow**: Large spread, low opacity gold-tinted shadows
5. **Status Indicators**: Colored dots (green/red) with subtle glow for live data

## Design Don'ts

- No blue/purple neon (this is gold, not cyberpunk)
- No rounded pill buttons (use defined rectangles with moderate radius)
- No white backgrounds anywhere
- No thin hairline fonts (everything should feel substantial)
- No playful illustrations (this is a serious trading platform)
- No gradient backgrounds (keep it flat and focused)
